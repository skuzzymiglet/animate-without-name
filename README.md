# animator

A command-line animation program (in the making!)

## Installation

`go get github.com/skuzzymiglet/animator`

## Usage

```
 animator [-h|--help]
           -e|--expr "<value>"
           -f|--files "<value>"
          [-f|--files "<value>" ...]
           -o|--out "<value>"
          [-r|--rate <float>]
```
